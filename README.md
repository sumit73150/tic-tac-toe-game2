# 🎮 Tic-Tac-Toe Game

A simple and interactive **Tic-Tac-Toe** game built using **HTML**, **CSS**, and **JavaScript**.

---

## 🕹️ Features

- Classic 2-player gameplay (X vs O)
- Intuitive UI with clean layout
- Win and draw detection
- Game reset functionality
